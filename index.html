Game of Throons

Game of Throons es un proyecto web estático desarrollado exclusivamente con HTML y CSS, ambientado en un universo de fantasía épica original.

El sitio presenta el reino de Throonia, sus casas nobles, personajes, historia, y secciones narrativas como privacidad, seguridad y contacto, todo integrado dentro del mundo.

Este proyecto tiene como objetivo reforzar los fundamentos del desarrollo web frontend, aplicando buenas prácticas de estructura semántica, organización del código y accesibilidad básica.
